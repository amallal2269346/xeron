Mint Authority

Frequently Asked Questions

A rug pull is a type of crypto scam where developers abandon a project and run away with investor funds. On Solana, this typically involves removing liquidity from a DEX pool, using mint authority to print new tokens and dump them, or enabling freeze authority to prevent investors from selling — all of which collapse the token price to zero.

Mint authority is a Solana SPL token permission that allows the wallet holding it to create (mint) additional tokens at any time. If a developer retains mint authority, they can inflate the supply infinitely, crashing the price. Legitimate projects either renounce mint authority or burn it to a null address.

A honeypot is a token you can buy but cannot sell. On Solana, this is typically achieved via freeze authority (which can block all transfers) or custom transfer instructions that revert on sell transactions. RugRadar checks for both patterns and flags them as critical risks.

A high risk score is a serious warning, but it's not financial advice. Some legitimate new projects may score moderately due to early-stage setup (e.g., liquidity not yet locked). Always verify flags manually, check the project's official channels, and use the score as one input among many in your research process.
